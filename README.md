# Matt Yu AI Portfolio Website

Live site: [https://ai.matt-yu.com](https://ai.matt-yu.com/)

This repository contains the source code for my AI-focused portfolio website. The site presents practical AI portfolio projects for job search, recruitment workflows, writing support, business analysis and productivity.

The AI portfolio is designed as a specialist subdomain of my main personal portfolio:

- Main portfolio: [https://matt-yu.com](https://matt-yu.com/)
- AI portfolio: [https://ai.matt-yu.com](https://ai.matt-yu.com/)

## Purpose

The purpose of this portfolio is to give recruiters and hiring teams a clear view of my practical AI project work as a junior AI, analyst, product analyst and operations analyst candidate.

The site focuses on structured prototypes, clear workflow design, local scoring logic, human review boundaries, project presentation and business value. It is intended to act as a hub page for my AI projects rather than replace the individual project repositories.

## Project Notes

- `index.html`, `style.css`, `script.js`, `assets/`, `CNAME` and `README.md` are the main files used by the live AI portfolio.
- `assets/thumbnails/` stores the GIF previews used in the AI project sections.
- `assets/Matt-Yu-Data-Analyst-CV.pdf` is linked from the landing section.
- `CNAME` preserves the custom domain for `ai.matt-yu.com`.
- The local folders for individual AI projects are ignored in this repository so this repo stays focused on the AI portfolio hub.
- The individual project websites and repositories are maintained separately.

## Build Transparency

This website was built as an AI-assisted portfolio project. I used Codex to help create and refine the HTML, CSS and JavaScript, while I focused on the content, structure, design direction and project presentation. I include this to be transparent about how the site was made and to avoid overstating my coding experience.
